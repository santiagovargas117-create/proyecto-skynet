Simulaciones de Colas - Skynet

Resumen del Proyecto

Este proyecto simula el uso de colas en sistemas inteligentes aplicados a seguridad, fabricación e inventario de partes de robots Skynet. Las colas permiten gestionar flujos de eventos, procesos y recursos de manera eficiente, modelando situaciones reales de la industria y la robótica.

  • Control de acceso: Simula la gestión de solicitudes de entrada a instalaciones seguras.
  • Línea de ensamblaje: Modela el paso de productos por distintas etapas de fabricación.
  • Inventario de partes: Administra el stock y consumo de componentes críticos de robots Skynet.

1. Control de Acceso (Seguridad)


            

                
                ¿Cómo funciona?
                    Cada persona solicita acceso y espera en una cola. El sistema procesa cada solicitud y decide si autoriza o deniega la entrada, simulando un control de seguridad real.

                ¿Cómo funciona?

Explicación de cada repuesto:

                
    
                    
  •  CPU: Es el cerebro del robot, encargado de procesar información, tomar decisiones y coordinar todas las funciones del sistema.
    • 
                    
  •  Sensor óptico: Permite al robot percibir su entorno mediante la detección de luz, colores, formas y movimientos, esencial para navegación y reconocimiento de objetos.
    • 
                    
  •  Actuador hidráulico: Proporciona movimiento y fuerza a las extremidades o mecanismos del robot, permitiendo realizar tareas físicas como levantar, empujar o ensamblar piezas.
    • 
                

                
Cada repuesto es gestionado en una cola de inventario, simulando el flujo y consumo de partes en la fabricación y mantenimiento de robots Skynet.

            

            

                
                ¿Cómo funciona?
                    Puedes modificar la cantidad inicial de cada repuesto. El sistema simula el consumo y muestra el inventario restante, gestionando el stock como en una fábrica real.
